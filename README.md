# Distributed_Systems_Projects

This repository contains implementations of various distributed systems and algorithms.

## Folders

### `block_chain`
A simple blockchain implementation demonstrating block creation, hashing, proof-of-work, and transaction validation.

### `bully_algorithm`
Python implementation of the Bully Algorithm for leader election in distributed systems.

### `chord_dht`
Distributed Hash Table using the Chord protocol for efficient key lookup in a peer-to-peer network.

### `real_time_forex_arbitrage_detector`
Detects real-time arbitrage opportunities in the foreign exchange market using graph-based algorithms.
